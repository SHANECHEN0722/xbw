# 2026 新年快乐 Demo

一个纯前端静态页面 Demo：`2026` 新年祝福 + 背景烟花（自动绽放）+ 底部“乌萨奇风格”小兔在看烟花 + 祝福弹幕（自动飘）+ 心形烟花 + 文字/图片粒子烟花（无第三方依赖）。

## 运行

建议用本地服务器打开：

```bash
python3 -m http.server 5173
```

然后访问：

- `http://localhost:5173`

## 玩法

- 点「抽一张」：随机祝福语
- 点「放烟花 / 心形烟花」或按空格：触发特效
- 在输入框写一句祝福，点「送上天空」：生成祝福弹幕 + 心形烟花
- 「自定义烟花」：输入短文字点「文字烟花」，或上传图片点「图片烟花」生成粒子形状爆炸
  - 文字/图片烟花都会先升空，再在空中炸开成对应形状
- 画面中央「拍立得」：未上传时默认显示 `img/image.png`，上传后可点「清除」恢复默认
